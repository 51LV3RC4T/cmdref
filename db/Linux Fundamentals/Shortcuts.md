Description :
    get a command line on virtual consoles . The CTRL key can be omitted if you are already in text mode, outside of Xorg or Wayland's graphical interface

Parameters : 

```cmd
Ctrl + Shift + F1   # F1 to F6  -> all will spawn terminal
```
